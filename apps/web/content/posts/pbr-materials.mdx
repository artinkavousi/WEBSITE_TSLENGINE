# Understanding PBR Materials

Physically-Based Rendering (PBR) creates realistic materials by simulating light physics.

## Key Parameters

- **Albedo**: Base color
- **Metalness**: 0=dielectric, 1=metal
- **Roughness**: 0=mirror, 1=matte
- **Normal**: Surface details

## Material Types

### Metals
```typescript
{ metalness: 1.0, roughness: 0.1 }
```

### Dielectrics
```typescript
{ metalness: 0.0, roughness: 0.5 }
```
