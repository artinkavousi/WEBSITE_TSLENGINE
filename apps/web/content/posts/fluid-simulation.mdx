## Implementation

### Velocity Advection

```wgsl
@compute @workgroup_size(8, 8)
fn advect(@builtin(global_invocation_id) id: vec3<u32>) {
  let coord = vec2<f32>(id.xy);
  let vel = velocity[coord];
  
  // Backtrace
  let prev = coord - vel * dt;
  
  // Bilinear interpolation
  velocity_out[coord] = sample(prev);
}
```
