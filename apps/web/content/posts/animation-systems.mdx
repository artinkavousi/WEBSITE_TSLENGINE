# Building Smooth Animation Systems

Animation brings your scenes to life.

## GSAP Integration

```typescript
import gsap from 'gsap';

gsap.to(mesh.position, {
  y: 2,
  duration: 1,
  ease: 'elastic.out(1, 0.5)',
});
```

## Easing Functions

- `power1/2/3/4`: Acceleration curves
- `back`: Overshoot effect
- `elastic`: Bouncy effect
- `bounce`: Gravity bounce

## State Machines

```typescript
const states = {
  idle: { scale: 1.0 },
  hover: { scale: 1.2 },
  active: { scale: 0.9 },
};

function transition(to) {
  gsap.to(object.scale, {
    ...states[to],
    duration: 0.3,
  });
}
```

## Performance Tips

- Use `will-change` for GPU acceleration
- Batch animations when possible
- Avoid layout thrashing
- Use `requestAnimationFrame`
